LaTeX style for Opal

There is a style file available for Opal programs which is designed to work with LaTeX 2e.

Documentation for opal2x.sty

Copy the following files from the doc/opal2x directory to some place where TeX will find them:

  • opal2x.sty
  • basicprog.sty
  • symbols.sty
  • scanner.sty
  • programsymbols.sty
  • tubprogram.sty
  • progfont.sty
  • slang.sty
